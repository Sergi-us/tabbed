# tabbed - generic tabbed interface

Container für Tab-Unterstützung in einbettbaren X-Anwendungen.

## Features

- XEmbed-Support
- Funktioniert mit surf, st, und anderen XEmbed-fähigen Programmen
- Minimalistisch und konfigurierbar

## Installation

```bash
git clone https://codeberg.org/Sergius/tabbed.git
cd tabbed
sudo make install
```

## Nutzung

```bash
# surf mit Tabs
tabbed -c surf -e

# st mit Tabs
tabbed -c st -w
```
